# Rishi Aryal | Professional Developer Portfolio

This is a **modern, fully responsive developer portfolio** showcasing my skills, projects, experience, and blog. Built for **Python, Django, DevOps, AI, and Full-Stack development**, this portfolio demonstrates professional industry-standard design and functionality, optimized for GitHub Pages.

## Features

- **Hero Section**: Animated introduction with typing effect highlighting roles and expertise.
- **About Me**: Professional summary and CV download.
- **Skills**: Technical skills with animated progress bars.
- **Projects**: Dynamic GitHub repository integration showing latest projects.
- **Experience & Education**: Timeline showcasing work history and qualifications.
- **Blog & Knowledge Base**: Interactive, filterable blog section with posts on Python, DevOps, AI, Full-Stack, and Cloud.
- **GitHub Contributions**: Live contribution graph integrated using GitHub Calendar.
- **Dark/Light Mode**: Toggleable theme for modern UI/UX.
- **Responsive Design**: Fully optimized for desktop, tablet, and mobile.

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript
- **Libraries**: Typed.js, GitHub Calendar
- **Backend**: None (static site)
- **Hosting**: GitHub Pages
